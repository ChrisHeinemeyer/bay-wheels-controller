Step 2: Configure WiFi

Hold the Flash button on your board, then connect via USB (or reset the board while holding the button). Release the button when you see the provisioning prompt.

Browser support: Chrome or Edge (desktop) required. Web Serial API is not supported in Safari or Firefox.
